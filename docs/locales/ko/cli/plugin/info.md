1. CLI 명령 실행에서 Shows을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
2. CLI 명령 실행에서 Displays을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
3. CLI 명령 실행에서 Lists을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
4. CLI 명령 실행에서 Shows, Python을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
5. CLI 명령 실행에서 Displays을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

## Use Cases

### 1. Check Plugin 호환성

```bash
# Verify plugin is compatible with your Truthound version
truthound plugins info my-validator
```

### 2. View Dependencies

```bash
# Check what dependencies a plugin needs
truthound plugins info custom-reporter
```

### 3. Export Plugin Info

```bash
# Export to JSON for documentation
truthound plugins info my-validator --json > plugin-info.json
```
