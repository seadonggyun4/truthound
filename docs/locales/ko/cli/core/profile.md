# truthound 프로파일

CLI 명령 실행에서 Generate을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

## Synopsis

```bash

```

## Description

CLI 명령 실행에서 `profile`을(를) 다루는 항목입니다:

- CLI 명령 실행에서 Basic, Statistics, Row을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
- CLI 명령 실행에서 Column, Statistics, Data을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
- CLI 명령 실행에서 Numeric, Analysis, Mean을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
- CLI 명령 실행에서 Distribution, Info, Skewness을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
- CLI 명령 실행에서 Pattern, Detection, Common을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

### Basic 프로파일링

```bash
truthound profile data.csv
```


```bash
truthound profile data.csv --format json -o profile.json
```

## Use Cases

### 1. Data Exploration

CLI 명령 실행에서 Quickly을(를) 다루는 항목입니다:

```bash
truthound profile unknown_data.csv
```

### 2. 데이터 품질 Baseline

CLI 명령 실행에서 Establish을(를) 다루는 항목입니다:

```bash
truthound profile baseline.csv --format json -o baseline_profile.json
```

### 3. Documentation Generation

CLI 명령 실행에서 Generate을(를) 다루는 항목입니다:

```bash
truthound profile production_table.parquet -o profile.json --format json
truthound docs generate profile.json -o data_docs.html
```

### 4. Pre-Analysis Check

CLI 명령 실행에서 Verify을(를) 다루는 항목입니다:

```bash
# Check for data issues
truthound profile analysis_data.csv

# Look for:
# - High null ratios
# - Unexpected unique counts
# - Outlier min/max values
```
