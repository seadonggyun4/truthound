## Minimal Quickstart

Start with a 태스크-only 플로우:

```python
from prefect import flow
from truthound_prefect import data_quality_check_task

@flow
async def validate_users(data):
    return await data_quality_check_task(
        data,
        rules=[{"column": "id", "check": "not_null"}],
    )
```

오케스트레이션 실행에서 Promote을(를) 다루는 항목입니다:

```python
from truthound_prefect import DataQualityBlock

block = DataQualityBlock(engine_name="truthound")
```

## Production Pattern

오케스트레이션 실행에서 Recommended을(를) 다루는 항목입니다:

1. 오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
2. saved `DataQualityBlock` for shared 설정
3. `create_quality_flow` or `quality_checked_flow` for standard 플로우 composition
4. 오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
