# 아키텍처

오케스트레이션 실행에서 Truthound, Orchestration을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

## Design Posture

오케스트레이션 실행에서 Truthound, `common`, Orchestration을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

## 아키텍처 At A Glance

```text
Host package
  -> host-native boundary (operator / resource / block / macro / script)
  -> shared runtime normalization
  -> engine creation and capability checks
  -> execution
  -> shared wire serialization
  -> host-specific metadata wrapping
```
