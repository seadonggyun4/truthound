## Local Commands

오케스트레이션 실행에서 Compile-oriented을(를) 다루는 항목입니다:

```bash
uv run --extra dbt python packages/dbt/scripts/run_first_party_suite.py \
  --mode compile \
  --target postgres
```

웨어하우스 execution verification:

```bash
uv run --extra dbt python packages/dbt/scripts/run_first_party_suite.py \
  --mode execute \
  --target postgres
```
