# Mage 문제 해결

## `io_config.yaml` Is Not Being Picked Up

오케스트레이션 실행에서 Check을(를) 다루는 항목입니다:

- the 파일 name is `io_config.yaml` or `io_config.yml`
- 오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
- 오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

## The Block Fails During 엔진 Creation

오케스트레이션 실행에서 Mage을(를) 다루는 항목입니다:

- 엔진 name
- 런타임 context assumptions
- missing 소스 metadata
- 오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

## The Quality 결과 Is Hard To Consume Downstream

오케스트레이션 실행에서 Mage을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.

## A 센서 Never Opens The Gate

오케스트레이션 실행에서 Check을(를) 다루는 항목입니다:

- 오케스트레이션 실행에서 `min_pass_rate`을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
