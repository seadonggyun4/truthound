## Example Shape

```yaml
default:
  TRUTHOUND_ENGINE: truthound
  TRUTHOUND_TIMEOUT: 300

data_sources:
  warehouse:
    type: postgres
    host: localhost
    port: 5432
    database: analytics
    username: ${PGUSER}
    password: ${PGPASSWORD}

  bronze_users:
    type: file
    path: data/bronze/users.parquet
    format: parquet
```
