## Minimal Quickstart

Load Mage 설정 explicitly:

```python
from truthound_mage import load_io_config

config = load_io_config("io_config.yaml", profile="production")
```

오케스트레이션 실행에서 Pair을(를) 다루는 항목입니다:

```python
from truthound_mage import CheckTransformer, CheckBlockConfig

transformer = CheckTransformer(
    config=CheckBlockConfig(
        rules=[{"column": "email", "check": "email_format"}]
    )
)
```

## Production Pattern

오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 다루는 항목입니다:

| 오케스트레이션 실행에서 Concern을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다. | 오케스트레이션 실행에서 Mage, Recommended, Boundary을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다. |
|--------|----------------------------|
| 소스 and sink connection 설정 | `io_config.yaml` plus 프로파일 selection |
| 시크릿 values | 오케스트레이션 실행에서 Mage을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다. |
| 검증 rules | block 설정 in 소스 control |
| 런타임 관측성 | 오케스트레이션 실행에서 Mage을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다. |

오케스트레이션 실행에서 Recommended을(를) 다루는 항목입니다:

- keep 시크릿 out of committed 설정 파일

- 오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
