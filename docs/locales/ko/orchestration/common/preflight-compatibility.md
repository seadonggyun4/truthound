## Recommended Usage Pattern

오케스트레이션 실행에서 관련 설정과 실행 흐름을(를) 다루는 항목입니다:

```python

request = EngineCreationRequest(
    engine_name="truthound",
    runtime_context=normalize_runtime_context(platform="airflow"),
)

report = run_preflight(request)
```
