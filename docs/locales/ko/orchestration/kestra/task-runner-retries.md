## Minimal Quickstart

Generated 플로우 accept 재시도 설정:

```python
from truthound_kestra import RetryConfig, generate_check_flow

yaml_content = generate_check_flow(
    flow_id="users_quality",
    namespace="production",
    retry=RetryConfig(max_attempts=3),
)
```
