# AI System Boundary

## Boundary Rules

1. AI 제안, 분석, 승인 흐름에서 `import truthound`을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
2. AI 제안, 분석, 승인 흐름에서 `truthound.core`, `truthound.ai`을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
3. AI 제안, 분석, 승인 흐름에서 `.truthound/ai/`을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
AI 제안, 분석, 승인 흐름에서 `.truthound/suites/`을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
4. AI 제안, 분석, 승인 흐름에서 관련 설정과 실행 흐름을(를) 기준으로 데이터 품질 검증, 워크플로우 자동화, 결과 해석 방법을 설명합니다.
