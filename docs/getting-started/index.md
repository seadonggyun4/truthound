# Getting Started

Welcome to Truthound! This section will help you get up and running quickly.

Truthound is designed to be easy to use while providing powerful data quality capabilities. In just a few minutes, you can:

1. Install Truthound
2. Validate your first dataset
3. Generate quality reports
