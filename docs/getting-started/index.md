# Getting Started

Truthound 3.0 keeps the easy path first:

1. Install the package and optional extras you need.
2. Start with `th.check()` or `truthound check`.
3. Let the framework create `.truthound/` and an auto-suite for you.
4. Move into `TruthoundContext`, `ValidationSuite`, planners, or plugins only when you need explicit control.
